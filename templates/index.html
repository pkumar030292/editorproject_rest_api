Generate Template â€“ Detailed Guide

The Generate Template tab allows you to create fully customized documents using your pre-defined templates. Follow these steps carefully to understand how the process works from template upload to final file download.

Step 1: Upload Your Template

- Navigate to the Upload Template tab.
- Select the file you want to use as a template. Templates can be in Text (.txt) or Word (.docx) format.
- For Text Files, placeholders must be written as {% raw %}{{PlaceHolder}}{% endraw %}. These placeholders will correspond to input fields on the web page.
- For Word Files, placeholders must be written as {% raw %}{{{PlaceHolder}}}{% endraw %}. These placeholders will also be converted into labels on the webpage for user input.

Step 2: Update Placeholders

- Before uploading, you can edit your template to ensure all dynamic content is represented by placeholders.
- Make sure each placeholder is unique and clearly named, as these will become input labels on the Generate tab.
- Once your template is ready, click Upload in the Upload Template tab. This will save your template to the system for further use.

Step 3: Template Availability

- After successful upload, your template will appear in the Generate Template dropdown menu.
- The system automatically detects all placeholders and maps them to input fields.
- Users can now select the template to generate customized files.

Step 4: Enter Placeholder Values

- Select your uploaded template from the dropdown menu in the Generate Template tab.
- All placeholders defined in the template will appear as input fields.
- Enter the desired values for each placeholder.
- You can either click Preview to see how the template will look with the entered data or click Generate to create the final file.

Step 5: File Generation

- Once the Generate button is clicked, the system automatically replaces all placeholders with the entered data.
- For text files, each {% raw %}{{PlaceHolder}}{% endraw %} is replaced by the corresponding input.
- For Word files, each {% raw %}{{{PlaceHolder}}}{% endraw %} is replaced while maintaining the formatting of the original document.
- The modified file is saved in the Generated Files directory with the current date and time appended to the filename.

Step 6: Download Generated Files

- Navigate to the Download tab.
- Your newly generated file will appear in the file list with a timestamp for easy identification.
- Click the filename to download it instantly. The downloaded file will contain all the user-provided placeholder values exactly as entered.

Additional Notes:

  • Ensure that all placeholders in your template are correctly named and enclosed in {% raw %}{{{PlaceHolder}}}{% endraw %} or {% raw %}{{PlaceHolder}}{% endraw %} depending on file type.
  • Templates can be updated anytime by re-uploading a new version. The system will automatically reflect the changes in the Generate tab.
  • Use the Preview option to verify your inputs and template formatting before generating the final file.
  • All generated files are timestamped to avoid overwriting and to maintain a history of all creations.
  • This workflow ensures a seamless transition from template design to file generation with minimal manual effort.

By following these steps, users can efficiently convert templates into fully customized documents in both text and Word formats, ensuring professional and accurate results every time.
